## Google 新闻采集为什么这么难搞

不是我一个人遇到这个问题。Google 对爬虫的检测机制这几年越来越激进——同一 IP 短时间内请求超过阈值，直接返回 429 或者 CAPTCHA 页面，根本拿不到数据。

自建代理池的成本也不低。买住宅 IP、维护轮换逻辑、处理 JavaScript 渲染……每一块单独拿出来都是工程量。ScraperAPI 把这些全打包成一个 API 端点，你只需要把目标 URL 传进去，它返回干净的 HTML 或结构化 JSON。

对 Google 新闻这个场景来说，它有一个专门的 Structured Data Endpoint，直接返回新闻标题、来源、发布时间、摘要，不用自己再写解析逻辑。这个功能我用得最多。
